About MacPulse

MacPulse is an automated static news aggregator designed to keep you updated with the latest in macOS, iOS, Apple Silicon, Vision Pro, and general Apple ecosystem leaks and rumors.

How it Works

Every hour, a background workflow executes on GitHub Actions. It scans popular Apple feeds configured in feeds.json (including MacRumors, 9to5Mac, AppleInsider, and Macworld), parses the freshest stories, runs regex-based auto-tagging, merges them into a single static articles.json, and publishes the database. This enables rapid load times via the GitHub Pages CDN.

Features

  • Optimized Apple Feeds: Dedicated feeds targeting core topics like Apple Silicon, iOS, macOS, and AR/VR.
  • Instant Client-Side Filtering: Instantaneous search and tab filtering built in JavaScript.
  • Private Bookmarks: Add stories to your bookmarks dashboard. Bookmarks are saved entirely in your local browser Cache (LocalStorage).
